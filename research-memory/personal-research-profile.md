# Personal Research Profile

Purpose: reusable preferences and habits that GeoCoScientist should consider across projects. Keep this concise and evidence-based.

## Stable Preferences

- Prefer Nature-level framing: one clear mechanism, one measurable response, one defensible inference design.
- Prefer turning broad topics into testable hypotheses with explicit treatment/driver, response, scale, and falsification tests.
- Prefer distinguishing static spatial patterns from dynamic temporal processes.
- Prefer pilot runs before full global reruns.
- Prefer documenting abandoned paths and why the framing changed.
- Prefer cautious causal language unless a counterfactual design is specified.

## Current Research Domains

- Protected areas, human footprint, halo/spillover, pressure concentration, and leakage risk.
- Forest degradation, carbon-water-climate feedbacks, LST/ET, restoration, and agroforestry.
- Remote sensing, spatial causal inference, land-use change, and ecological resilience.

## Decision Habits To Reuse

- If a leakage framing is weak, test whether the stronger story is boundary gradient, protective halo, or pressure-concentration edge.
- If a sample filter is inherited from an earlier chapter, audit whether it still matches the current manuscript target.
- If a global average is weak, inspect heterogeneity by region, biome, baseline pressure, age, governance, terrain, and land-use context.

## Update Rule

Only update this file after an explicit user request such as "synchronize GeoCoScientist memory" or "update my research profile". Add a short evidence note when possible.
